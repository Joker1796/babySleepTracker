# Режим малыша

Статический сайт + PWA для отслеживания режима ребёнка (0–12 месяцев): сон, прогулки,
купание, лекарства — и умные подсказки, когда укладывать в следующий раз, как
корректировать режим и что делать, если малыш не засыпает.

## Возможности

- Несколько детей, переключение в одно касание
- Кнопка «Уснул/Проснулся» с живым таймером; прогулка, купание, лекарства
- Движок подсказок: прогноз следующего сна по окну бодрствования с учётом
  короткого предыдущего сна, прогноз ночного укладывания, ~20 правил-советов
  (перегул, дефицит дневного сна, регрессы, переходы между количеством снов…)
- Календарь: врачи, прививки, измерения, зубы — и свои планы с отметкой
  выполнения; кнопки-шаблоны планов задаются в настройках, план добавляется
  на день в один тап, планы на сегодня видны на главном экране
- Режим «Болезнь»: температура, лекарства с напоминаниями, дни болезни
  подсвечиваются в календаре
- Свой режим дня: окна бодрствования и число снов вручную + распорядок дня
  с учётом запланированных событий
- База знаний: возрастные нормы сна и 20 статей (ритуалы, самостоятельное
  засыпание, ночные пробуждения, безопасность сна и др.), фильтр по возрасту
- История по дням с редактированием событий, статистика сна за 7/14 дней
- Данные только на устройстве (IndexedDB) + экспорт/импорт JSON
- Полноценная PWA: офлайн-режим, установка на домашний экран, тёмная тема

## Команды

```bash
npm install        # зависимости
npm run dev        # dev-сервер
npm test           # тесты движка подсказок (vitest)
npm run build      # сборка в dist/ (статика, хостить где угодно)
npm run preview    # локальный просмотр сборки
npm run icons      # перегенерировать PNG-иконки PWA
```

## Архитектура

- `src/data/` — база знаний как данные: нормы сна по возрастам, правила подсказок,
  статьи, реестр типов событий (добавить «кормление» — одна запись в `eventTypes.js`)
- `src/logic/` — чистые функции: возраст, анализ сна, движок рекомендаций (покрыты тестами)
- `src/stores/` — Pinia-сторы поверх Dexie (IndexedDB)
- `src/views/`, `src/components/` — Vue 3, mobile-first, без UI-библиотек

Подсказки носят информационный характер и не заменяют консультацию педиатра.
